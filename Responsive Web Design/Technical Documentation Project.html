Introduction

In this documentation, we will discuss CSS properties that corresponds to element positioning. This is fairly easy in CSS, so you shouldn't take long to understand it. Main properties are:

  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky

Static

Static position does nothing in CSS. It sits where it sits. You can change position by margin property, but for dinamic layout, it is almost useless, so we don't recomend it.

position: static;

It has wide support in all major browsers

Relative

Relative positioning is most useful property in that list. When you give element relative positioning, it becomes relative to its parent element, so you can make positioning whith left, right, top and bottom relative to its parent

position: relative;
top: 10px;
left: 20px;

It has wide support in all major browsers

Absolute

Absolute positioning sets element to its root element, mainly html, so absolutly positioned element is relative to body, not its parent div.

position: absolute;
top: 500px;
left: 300px;

It has wide support in all major browsers

Fixed

Fixed positioning sets element relative to viewport, so when you scroll, it isn't scrolled, but sits where you position it

position: fixed;
right: 20px;
bottom: 100px;

It has wide support in all major browsers

Sticky

Sticky property is hybrid between static and fixed. Until it reaches point, where you set it, it scrolls, but after that, it becomes fixed.

position: sticky;
top: 50px;
right: 20px;

It has wide support in all major browsers
